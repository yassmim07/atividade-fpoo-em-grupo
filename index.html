O que é css e para que serve?

O Cascading Style Sheets (CSS) é uma "folha de estilo" composta por “camadas” e utilizada para definir a apresentação (aparência) em páginas da internet que adotam para o seu desenvolvimento linguagens de marcação (como XML, HTML e XHTML). O CSS define como serão exibidos os elementos contidos no código de uma página da internet e sua maior vantagem é efetuar a separação entre o formato e o conteúdo de um documento.



É importante lembrar que..

Para estilizarmos uma página html no css, é necessário chamar o arquivo css dentro da página do html

É simples fazer isso, o seu arquivo style.css provavelmente vai estar dentro de uma pasta chamada "css", basta apenas conectar esses arquivos um no outro, usando a tag <(link rel)>






Basicamente, o CSS serve para estilizar todo o conteúdo dentro das tags do html e dar um design melhor para a página.

Exemplo do body e da tag <(p)> sendo estilizadas no css, sendo definido a cor, tamanho das letras, fonte etc. Nesse caso, a cor que foi definida nesse exemplo vai ficar na página toda. E toda a tag <(p)> no html vai ficar com essa formatação que foi definida no css.




Mas..e se depois eu quiser estilizar de uma maneira diferente apenas uma tag <(p)>? e deixa-lá com um padrão diferente do padrão que coloquei?

Isso é mais simples do que parece, basta criar uma class.

Class é um atributo global utilizado nas linguagens para mudar um elemento em especifico. No html, basta puxar esse comando dentro da tag em especifico que você deseja alterar.




No css, basta chamar essa class usando o "." e em seguida colocando seu nome. Pronto! uma tag em especifico foi estilizada usando a class.
